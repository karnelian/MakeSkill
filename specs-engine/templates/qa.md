## QA 프로세스

### 1. 기능 테스트
- P0 기능: 전체 시나리오 테스트
- P1 기능: 핵심 플로우 테스트
- 엣지 케이스 테스트

### 2. 성능 테스트
- 응답 시간 측정 (목표 대비)
- 동시 접속 부하 테스트
- 메모리/CPU 사용량 모니터링

### 3. 호환성 테스트
- 유형별:
  - 웹: 브라우저 호환성 (Chrome/Firefox/Safari/Edge)
  - 모바일: iOS/Android 버전별
  - 게임: 디바이스 성능별
  - API: SDK 버전별
  - 데스크톱: OS 버전별

### 4. 리그레션 테스트
- 이전 마일스톤 기능 재검증
- 수정된 버그 재발 여부
