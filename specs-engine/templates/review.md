## 리뷰 체크리스트

### 1. 설계 정합성
- API 설계 ↔ 실제 엔드포인트 매핑
- DB 스키마 설계 ↔ 실제 마이그레이션 매핑
- 보안 설계 ↔ 실제 인증/인가 구현

### 2. 코드 품질
- 코딩 컨벤션 준수
- 에러 핸들링 적절성
- 중복 코드 여부
- 테스트 커버리지

### 3. 보안 리뷰
- OWASP Top 10 체크
- 인증/인가 취약점
- 입력 검증
- 민감 데이터 노출

### 4. 성능 리뷰
- N+1 쿼리
- 불필요한 렌더링
- 메모리 누수 가능성
- 캐싱 적절성
