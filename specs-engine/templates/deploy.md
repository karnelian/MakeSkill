## 유형별 배포

### 웹 SaaS / API 서비스
- CI/CD 파이프라인 검증
- Blue-Green / Canary 배포 전략
- 롤백 계획

### 모바일 앱
- App Store / Play Store 심사 제출
- 메타데이터 + 스크린샷 준비
- 리뷰 가이드라인 준수 확인

### 게임
- 스토어 심사 제출
- 서버 인프라 스케일링
- 소프트론칭 지역 설정

### 데스크톱 앱
- 코드 서명 + 공증
- 인스톨러 빌드
- 자동 업데이트 서버

### 크롬 확장
- 웹스토어 심사 제출
- 개인정보처리방침 게시
- 권한 최소화 확인
