## 프로세스

### 1. 마일스톤 선택
- STATE.md에서 현재 마일스톤 확인
- 해당 마일스톤의 태스크 파일 로드

### 2. 태스크 실행
- 태스크를 순서대로 실행 (의존성 순서 준수)
- 각 태스크 완료 시 상태 업데이트: ⬜ → 🔄 → ✅
- 블로커 발생 시 기록 + 우회 방안 탐색

### 3. 게이트 체크
- 마일스톤 완료 기준 확인
- 전체 태스크 ✅ 여부
- 블로커 0건 여부

### 4. 다음 마일스톤 전환
- 현재 마일스톤 완료 → STATE.md 업데이트 → 다음 마일스톤 안내
